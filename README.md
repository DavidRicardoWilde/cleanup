# Raycast Extension -- Cleanup

A Raycast extension to help you clean up your Mac or Windows system. Easily scan and uninstall applications, remove installer files, and clean up development workspace directories (like node_modules, target, etc.).

## Features
- Scan and uninstall applications completely
- Clean up installer files
- Clean up development project build/cache directories
- Cross-platform: macOS & Windows support

## Getting Started
1. Install dependencies:
	```sh
	pnpm install
	```
2. Start development:
	```sh
	pnpm dev
	```
3. Build for production:
	```sh
	pnpm build
	```
