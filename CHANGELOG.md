# Cleanup Changelog

## [0.1.0] - 2026-01-19

### Added
- **Cleanup Apps**: Scan and uninstall applications completely from your system
- **Cleanup Installers**: Scan and remove installer files (.dmg, .pkg, .mpkg, .iso, .xip, .zip) from common download locations
- **Cleanup Workspaces**: Detect and clean up build/cache directories (node_modules, target, dist, etc.) across development projects
